# Login Page

A clean and well-structured login page UI created using HTML. This project serves as a starting point for integrating user authentication interfaces into web applications.

## Features

* **User Authentication Fields**: Input fields for Username and Password.
* **Icon Integration**: Utilizes [Boxicons](https://boxicons.com/) for intuitive visual cues (user and lock icons).
* **Additional Options**: Includes a "Remember me" checkbox and a "Forgot password?" link.
* **Registration Prompt**: Directs new users to a registration page ("Don't have an account? register here").
* **Clean Structure**: Well-organized HTML5 layout ready for custom CSS styling.

## Technologies Used

* **HTML5**: Semantic structure of the webpage.
* **CSS**: For styling (links to an external `style.css` file).
* **Boxicons**: External library for UI icons.

## Getting Started

### Prerequisites

You just need a modern web browser to view the login page. No server setup is required for viewing the frontend UI.

### Installation and Usage

1.  Clone the repository or download the source code.
2.  Ensure that `index.html` and your custom `style.css` are in the same directory.
3.  Double-click on `index.html` to open it in your default web browser.

## File Structure

* `index.html`: The main HTML document containing the login form structure.
* `style.css`: The stylesheet (you will need to create and populate this file to add your custom designs).
